# AI Code Reviewer

An AI-powered Code Reviewer built using Flask and Google's Gemini API. It analyzes Python code, detects bugs, suggests improvements, and recommends best coding practices.

## Features

- AI-powered code review
- Bug detection
- Code optimization suggestions
- Clean and responsive interface
- Flask backend
- Google Gemini API integration

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini API

## Installation

```bash
git clone https://github.com/prashansaucer/AI-Code-Reviewer.git
cd AI-Code-Reviewer
pip install -r requirements.txt
python app.py
```

Then open your browser and visit:

```
http://127.0.0.1:5000
```
